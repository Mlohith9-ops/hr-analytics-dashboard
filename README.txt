HR Analytics Dashboard (Power BI)
This project is a simple HR Analytics dashboard created using **Power BI**.  
The goal was to understand employee attrition, demographics, and overall workforce distribution using a real HR dataset.

What This Dashboard Shows->

Attrition Rate:  
Percentage of employees who have left the company.

Total Employees: 
Total number of people currently in the company.

Gender Distribution:  
A donut chart showing the split between male and female employees.

Department-wise Attrition:  
Which departments are losing the most employees.

Age Group Distribution:  
How employees are spread across different age groups.
